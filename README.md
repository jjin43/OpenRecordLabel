# SUI-Hackathon
Blockchain project for ASU/ SUI-Hackathon  
End-to-End dApp for artists
Artists can list music using cryptocurrency on the webapp, music stored using IPFS.

# Dependencies:
NodeJS  
Sui Client  
NFT.Storage - IPFS API
Supabase - DB as Service  

## How to Run
```
# to start server
npm install
npm start

# To stop server
npm stop

```
